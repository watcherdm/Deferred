Deferred by webspinner

jQuery Deferred library for node js

Just a port of the popular jQuery Deferred object and acompanying when method to node js so I could use a consistent object construction across server and browser.

Dependencies

vows for running the tests. node js for using the library as packaged. npm if you want to install it all nifty like.

Install

npm install Deferred
